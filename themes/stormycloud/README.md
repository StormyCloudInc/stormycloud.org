# My Hugo Theme

## Overview
This is a custom Hugo theme designed for StormyCloud Inc. It provides a clean and modern layout for showcasing content related to digital privacy and security.

## Installation
To use this theme in your Hugo site, follow these steps:

1. **Clone the Theme**: 
   Clone this repository into the `themes` directory of your Hugo site.
   ```
   git clone https://github.com/yourusername/my-hugo-theme.git themes/my-hugo-theme
   ```

2. **Update Configuration**: 
   In your site's `config.toml`, set the theme to `my-hugo-theme`:
   ```toml
   theme = "my-hugo-theme"
   ```

3. **Add Content**: 
   Create new content using the archetypes provided in the theme. For example:
   ```
   hugo new post/my-first-post.md
   ```

4. **Run Hugo**: 
   Start the Hugo server to see your site in action:
   ```
   hugo server
   ```

## Features
- **Responsive Design**: The theme is built to be mobile-friendly and responsive.
- **Customizable**: Easily modify styles and layouts to fit your needs.
- **SEO Optimized**: Includes meta tags and structured data for better search engine visibility.

## Directory Structure
- `archetypes/`: Contains default content templates.
- `assets/`: For storing images, fonts, and other assets.
- `layouts/`: Contains HTML templates for different content types.
- `static/`: For static files like CSS and images.
- `theme.toml`: Metadata about the theme.
